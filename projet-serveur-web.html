Projet — Déploiement Serveur Apache

Installation et configuration d'un serveur web Linux avec Apache

Déploiement d'un serveur web Apache sous Linux incluant la configuration des hôtes virtuels, du SSL/TLS, des modules et de la sécurité.

🎯 Objectifs du projet

  • Installer et configurer Apache2 sur un serveur Linux.
  • Créer des hôtes virtuels (Virtual Hosts).
  • Activer le SSL/TLS avec certificats auto-signés.
  • Configurer les modules nécessaires (PHP, mod_rewrite, etc.).
  • Implémenter des règles de sécurité (firewall, permissions).

🏗️ Architecture du serveur

Clients HTTP/HTTPS ↓ Pare-feu (Ports 80, 443) ↓ Serveur Apache | ├── VirtualHost 1 (exemple.com:80) ├── VirtualHost 2 (exemple.com:443 SSL) └── Module PHP, mod_rewrite ↓ Fichiers web (/var/www/html)

⚙️ Étapes de déploiement

1 · Installation d'Apache

  • Installation : apt install apache2.
  • Démarrage du service : systemctl start apache2.
  • Activation au démarrage : systemctl enable apache2.
  • Vérification du statut.
Apache2 Linux (Ubuntu) Systemctl

2 · Configuration de base

  • Édition du fichier /etc/apache2/apache2.conf.
  • Configuration du nom de serveur et du port d'écoute.
  • Tests de configuration : apache2ctl configtest.
Configuration Apache Fichiers conf Terminal

3 · Hôtes virtuels (Virtual Hosts)

  • Création de fichiers .conf dans /etc/apache2/sites-available.
  • Activation avec a2ensite nom-site.
  • Création des répertoires web : /var/www/site1, /var/www/site2.
  • Configuration DNS (hosts ou DNS réel).
Virtual Hosts Domaines DNS

4 · SSL/TLS et HTTPS

  • Génération d'un certificat auto-signé : openssl req.
  • Activation du module SSL : a2enmod ssl.
  • Configuration des Virtual Hosts SSL (port 443).
  • Redirection HTTP → HTTPS (optionnel).
OpenSSL Certificats SSL/TLS

5 · Activation des modules et sécurité

  • Activation de mod_rewrite : a2enmod rewrite.
  • Activation de mod_php : apt install libapache2-mod-php.
  • Configuration des droits des fichiers : chown www-data:www-data.
  • Ouverture des ports dans le pare-feu : ufw allow 80,443.
Modules Apache Permissions Firewall
