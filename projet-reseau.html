Projet — Infrastructure Réseau

Mise en place d'un serveur Active Directory, DNS et DHCP sous Windows Server

Projet réalisé dans le cadre du BTS SIO SISR consistant à concevoir et déployer une infrastructure réseau complète en environnement virtualisé : contrôleur de domaine Active Directory, serveur DNS, serveur DHCP et gestion des utilisateurs.

🎯 Objectifs du projet

  • Déployer un contrôleur de domaine Active Directory sous Windows Server.
  • Configurer un serveur DNS pour la résolution de noms au sein du domaine.
  • Mettre en place un serveur DHCP pour l'attribution automatique des adresses IP.
  • Créer et gérer des comptes utilisateurs, groupes et unités d'organisation (OU).
  • Appliquer des stratégies de groupe (GPO) pour uniformiser les environnements utilisateurs.
  • Intégrer des postes clients Windows au domaine.

🏗️ Architecture mise en place

🌐 Réseau local (LAN) │ ├── 🖥️ Serveur Windows Server 2019 │ ├── Contrôleur de domaine (AD DS) │ ├── Serveur DNS → résolution interne │ └── Serveur DHCP → plage IP automatique │ ├── 💻 Poste client 1 (Windows 10) — joint au domaine └── 💻 Poste client 2 (Windows 10) — joint au domaine

Infrastructure déployée en environnement virtualisé (VMware Workstation)

⚙️ Étapes de réalisation

1 · Installation et configuration de Windows Server

  • Installation de Windows Server 2019 en machine virtuelle.
  • Configuration de l'adresse IP statique du serveur.
  • Renommage du serveur et paramétrage initial.
Windows Server 2019 VMware Workstation

2 · Déploiement d'Active Directory (AD DS)

  • Installation du rôle AD DS via le Gestionnaire de serveur.
  • Promotion du serveur en contrôleur de domaine.
  • Création du domaine (ex : nesrine.local).
  • Création des Unités d'Organisation (OU) : Utilisateurs, Groupes, Ordinateurs.
  • Création de comptes utilisateurs et attribution aux groupes appropriés.
Active Directory AD DS OU Utilisateurs & Groupes

3 · Configuration DNS et DHCP

  • Configuration des zones DNS directe et inverse pour le domaine.
  • Création des enregistrements A et PTR pour les hôtes.
  • Configuration du serveur DHCP avec une plage d'adresses dédiée.
  • Exclusion des adresses statiques (serveur, imprimantes…).
DNS DHCP Zones DNS Enregistrements A/PTR

4 · Stratégies de groupe (GPO)

  • Création de GPO pour uniformiser le fond d'écran des postes du domaine.
  • Restriction de l'accès au Panneau de configuration pour les utilisateurs standard.
  • Déploiement de paramètres de sécurité (verrouillage de session automatique).
GPO Group Policy Sécurité

5 · Intégration des postes clients au domaine

  • Configuration des postes clients pour pointer vers le serveur DNS du domaine.
  • Jonction des postes Windows 10 au domaine Active Directory.
  • Vérification de l'authentification des utilisateurs du domaine sur les postes clients.
  • Test d'application des GPO sur les sessions utilisateurs.
Windows 10 Jonction domaine Authentification AD
